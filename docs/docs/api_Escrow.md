<div class="contract-doc"><div class="contract"><h2 class="contract-header"><span class="contract-kind">contract</span> Escrow</h2><p class="description">A contract for managing trading of funds between buyers and sellers. This contract manages all transactions between buyers and sellers in the marketplace. When a buyer commits to purchasing a listing, a new transaction is created and buyer&#x27;s purchase funds are held in this Escrow contract until the buyer confirms successful delivery of the item from seller. New transactions are started by creating a new Transaction struct and storing it in the map. Only 1 transaction per listing can be maintained at a time. Minimal safety checks are implemented here to avoid double payments or completing trades before payment is actually received in escrow.</p><div class="source">Source: <a href="https://github.com/ks510/trade-now/blob/v1.0.0/contracts/Escrow.sol" target="_blank">contracts/Escrow.sol</a></div><div class="author">Author: Karen Suen</div></div><div class="index"><h2>Index</h2><ul><li><a href="Escrow.html#confirmItemReceived">confirmItemReceived</a></li><li><a href="Escrow.html#fundTransaction">fundTransaction</a></li><li><a href="Escrow.html#getAllUserTransactions">getAllUserTransactions</a></li><li><a href="Escrow.html#getTransaction">getTransaction</a></li><li><a href="Escrow.html#getTransactionAmount">getTransactionAmount</a></li><li><a href="Escrow.html#getTransactionBuyer">getTransactionBuyer</a></li><li><a href="Escrow.html#getTransactionSeller">getTransactionSeller</a></li><li><a href="Escrow.html#getTransactionStatus">getTransactionStatus</a></li><li><a href="Escrow.html#isAwaitingDelivery">isAwaitingDelivery</a></li><li><a href="Escrow.html#isAwaitingPayment">isAwaitingPayment</a></li><li><a href="Escrow.html#startTransaction">startTransaction</a></li></ul></div><div class="reference"><h2>Reference</h2><div class="modifiers"><h3>Modifiers</h3><ul><li><div class="item modifier"><span id="isAwaitingDelivery" class="anchor-marker"></span><h4 class="name">isAwaitingDelivery</h4><div class="body"><code class="signature">modifier <strong>isAwaitingDelivery</strong><span>(uint _id) </span></code><hr/><div class="description"><p>Throws error if transaction is not in AWAITING_DELIVERY status.</p></div><dl><dt><span class="label-parameters">Parameters:</span></dt><dd><div><code>_id</code> -
